UI COMPONENTS ASSIGNMENT - A Guide to Our Game

After running the Makefile, run the command `bin/myGame` to run our game.

Press Q to switch between a magic and non-magic shield. The magic shield will deflect mage attacks.

Lastly, press the escape key to open the menu.
